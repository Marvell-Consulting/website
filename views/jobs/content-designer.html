Content Editor - Contract (outside IR35)

Summary of the work

Marvell Consulting is currently building an online directory of data and information standards for the NHS. The purpose of the directory is to signpost users to the standards required to collect and share information consistently.

The directory is not yet live. It is currently being populated with standards and we need an experienced content editor to oversee this process. This includes ensuring content is clear, accurate and correctly tagged to the appropriate categories.

Once the site is live you will be the main person responsible for keeping the site up to date. This includes by:

  • liaising with stakeholders to keep their standard pages up to date
  • managing an inbox of queries from people using the directory
  • using site monitoring tools and user feedback to suggest improvements to standard pages and the wider directory
  • helping draft internal processes for keeping the site up to date

You may also be called upon to help improve the design of the front-end interface in response to user feedback - for example, by improving page titles or other content. Content or UX design experience is therefore beneficial although we can offer coaching in this area.
